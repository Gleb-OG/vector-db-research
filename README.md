# db-models-research
## Техническое задание №1
С помощью vector_db_research.pdf можно ознакомиться с практической частью работы в удобном формате.

Также в репозитории можно найти оригинал ноутбука с выполненным заданием: vector_db_research.ipynb. Стоит учесть, что файл запускался в Google Collab, в среде выполнения которого хранился api-ключ для pinecone, поэтому для самостоятельного запуска эксперимента требуется подключить собственный ключ.  

В файле task_data.json хранится база фактов и запросов для выполнения практической части.
Факты были разделены на 10 тем:
1. temperature
2. humidity
3. raw_material
4. vibration
5. coolant
6. robots
7. inventory
8. refrigeration
9. packaging
10. delivery
    
По одному запросу на каждую тему (группу из 10 "очевидных" для него фактов).
